###东北大学秦皇岛分校数学与统计学院毕业生管理系统前端

**介绍**

基于react react-router react-bootstrap redux yeoman的管理系统

**Install**

----
```javascript

1. git clone https://github.com/iQoderi/st-graduate-management-fe.git

2. npm install

```

**Run**

----
```javascript
1. npm start

```

**dist**

`npm run dist`

**确保你的电脑已经装好了webpack webpack-dev-server**
